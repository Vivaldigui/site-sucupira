## Publicação

Os posts têm datas futuras entre 01/07/2026 e 30/07/2026. A listagem, as rotas públicas e o sitemap usam filtro por data de publicação no fuso `America/Sao_Paulo`.

Para publicar automaticamente no Firebase, é preciso executar um build e deploy todos os dias. O workflow incluído neste projeto faz build diário sem secrets. Para deploy automático, configure manualmente um secret de autenticação do Firebase no GitHub e adicione a etapa de deploy conforme a política do repositório.

Comando manual validado para este projeto:

```powershell
npm.cmd run build
npx.cmd firebase-tools@latest deploy --only hosting --project blog-sucupira --non-interactive
```
